## hw3：判斷質數
主要是 return true 有時會下錯位置。第一次寫時下在 for loop 裡面～ 因為直覺寫法為 if 條件後，要接著 else。

這邊寫一次我的理解，以 isPrime(5) 為例：
1. 帶進 for loop
	1. 跑第三行 i = 2; i < 3
	2. 跑第四行 2 % 3 餘數不等於 0，
	3. i ++ -> i = 3
	4. 再跑一次上述步驟...到 5 不小於 5，所以跳出整個 for loop。
2. 既然無法整除的數，就是質數了 -> return true

	var newA = parseInt(a)
